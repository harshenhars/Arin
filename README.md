This repository contains a reproducible Python workflow for automated bulk downloading of ALOS PALSAR RTC_HI_RES products from ASF Data Search (Vertex). The workflow runs in Google Colab and stores outputs directly in mounted Google Drive directories.

<b> Key Features </b>
<li> Integration with ASF bulk download scripts </li>
<li> Google Drive mounting in Colab </li>
<li> Automatic duplicate file detection </li>
<li> Earthdata authentication support </li>
<li> Scalable for large regional downloads </li>
<br> <br> <br>
<b> Use Cases </b>

<li> DEM generation </li>
<li> Flood hazard mapping </li>
<li> SAR-based terrain analysis </li>
<li> Bangladesh-wide RTC data collection </li>
